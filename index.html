Congratulations!

If you are reading this in your web browser, then the only logical conclusion is that the http://chronos.dev/ virtualhost was setup correctly. :)

You can find the configuration file for this virtual host in:
/private/etc/apache2/virtualhosts/chronos.dev

You will need to place all of your website files in:
/Library/WebServer/Documents/chronos

For the latest version of this script, tips, comments, donations, etc. visit:
http://patrickg.com/virtualhost
